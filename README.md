# Vue 3 + Vite
This is the HIS's Client using Vue3 and SpringBoot.
You can run by these way.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
